AI Drug Screening

Modelo matemático + explicación intuitiva

Introducción

Las curvas dosis–respuesta permiten estudiar cómo cambia el efecto de un fármaco al modificar la concentración.
Este tipo de curva describe todo el comportamiento del sistema biológico, lo que permite entender mejor la eficacia del fármaco frente a métodos más simples.

Justificación

Se utilizan curvas completas en lugar de valores como el IC50 para tener una visión más real del comportamiento del fármaco.
Dos fármacos pueden tener el mismo IC50 pero efectos completamente distintos en otras zonas de la curva, lo que hace necesario analizar toda la relación dosis-respuesta.

Problema técnico

Se dispone de datos experimentales y se busca una función que los represente.
Este problema es un sistema sobredimensionado: muchos datos y pocos parámetros, por lo que se necesita ajuste de curvas.

Modelo matemático

Se usa una función logística: f(c) = Emax / (1 + (c/IC50)^h)
Los parámetros se calculan minimizando el error entre la curva teórica y los datos experimentales mediante mínimos cuadrados.

Interpretación

El modelo permite analizar cómo varía el efecto con la dosis.
Esto ayuda en la selección de dosis óptimas y en la evaluación de candidatos farmacológicos, facilitando la toma de decisiones incluso sin conocimientos matemáticos avanzados.
