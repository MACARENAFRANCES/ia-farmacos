Modelo matemático intuitivo

Conceptos clave

Proceso
La inteligencia artificial utiliza curvas dosis–respuesta completas porque permiten analizar mejor el comportamiento del fármaco frente a usar únicamente el IC50, proporcionando más información sobre eficacia y respuesta. Se tienen datos experimentales de concentración y respuesta. El problema consiste en encontrar una función matemática que se ajuste a estos datos. Es un sistema sobredimensionado.
Modelo matemático
Se utiliza una función logística: f(c) = Emax / (1 + (c/IC50)^h) Los parámetros se ajustan mediante mínimos cuadrados para minimizar el error con respecto a los datos.
Interpretación
El modelo permite entender cómo cambia el efecto con la dosis, ayudando a seleccionar concentraciones óptimas y evaluar la eficacia de un fármaco incluso sin conocimientos matemáticos avanzados.
