Modelización matemática + explicación intuitiva

Justificación

Pasa el ratón para ver explicación
La inteligencia artificial moderna no usa solo valores como el IC50 porque estos simplifican demasiado el comportamiento de un fármaco. Se prefieren curvas completas dosis–respuesta porque permiten analizar: • cómo crece el efecto • si hay saturación • el rango útil de dosis Desde el punto de vista matemático, esto implica ajustar una curva a muchos datos → sistema sobredimensionado.

Problema técnico

Pasa el ratón 👇
Se tienen datos experimentales (concentración vs respuesta). Queremos encontrar una función que describa ese comportamiento de forma aproximada. Esto lleva a estimar parámetros del modelo usando técnicas numéricas.

Modelo matemático

Pasa el ratón 👇
Se usa una función logística: f(c) = Emax / (1 + (c/IC50)^h) Los parámetros se ajustan minimizando el error entre datos reales y modelo teórico usando mínimos cuadrados.
