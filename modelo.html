Predicción de curvas dosis–respuesta con IA

Este trabajo analiza el uso de inteligencia artificial para predecir curvas dosis–respuesta completas en cribado farmacológico, como alternativa al uso exclusivo del IC50.

1. Justificación técnica

Trabajos recientes destacan que una curva completa proporciona más información que un valor aislado, permitiendo analizar eficacia, saturación y comportamiento farmacológico completo.

2. Problema técnico

Se dispone de múltiples observaciones experimentales (m datos) y se buscan pocos parámetros (4), lo que genera un sistema sobredimensionado.

Modelo:

f(c) = Emin + (Emax - Emin) / (1 + (c / IC50)^h)

3. Modelo matemático

Se define el problema como minimización de errores:

min Σ (yi - f(ci))²

Se pueden usar versiones robustas (Huber) para evitar influencia de valores atípicos.

4. Resolución numérica

Se emplean métodos iterativos como:

  • Gauss-Newton
  • Levenberg-Marquardt

Estos métodos ajustan progresivamente los parámetros mediante la matriz jacobiana.

5. Discusión matemática

  • Identificabilidad del modelo
  • Condicionamiento numérico
  • Heterocedasticidad de datos
  • Robustez frente a outliers

6. Impacto farmacológico

Permite analizar mejor candidatos, comparar fármacos y mejorar la selección en cribados.

7. Conclusión

Este enfoque combina:

  • IA
  • Sistemas sobredimensionados
  • Ajuste de curvas
  • Optimización numérica
