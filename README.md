# IA en cribado farmacológico

Este proyecto consiste en el desarrollo de una página web interactiva para explicar el uso de modelos matemáticos en farmacología computacional, en particular el ajuste de curvas dosis–respuesta.

## Objetivo

Mostrar cómo la inteligencia artificial y los métodos numéricos permiten estimar parámetros farmacodinámicos a partir de datos experimentales.

## Contenido

- Simulador interactivo de curva dosis–respuesta
- Calculadora IC50
- Explicación intuitiva del modelo
- Desarrollo matemático detallado en una página adicional

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript
- GitHub Pages (deploy de la web)

## Modelo matemático

Se utiliza una función logística del tipo:

f(c) = Emax / (1 + (c/IC50)^h)

El problema se resuelve mediante mínimos cuadrados en un sistema sobredimensionado.
